# 🌟 EduSpark – Smart Campus Resource Management System

## 🔗 Skill Exchange & Smart Campus Platform for University Students

EduSpark is a **Smart Campus web application** developed for the IT3030 – PAF Assignment.  
It enables efficient **facility management, booking, and collaboration**, while ensuring **accountability, structured workflows, and real-time system operations**.

---

## 📌 What EduSpark Promotes

- 📚 Peer Learning  
- 🤝 Responsible Collaboration  
- ⏱️ Time-based Task & Resource Usage  
- 📈 Skill & System Efficiency Growth  

---

## ❗ Problem Statement

Many university systems face challenges such as:

- ❌ Lack of structured resource management  
- ❌ Manual booking processes  
- ❌ No real-time availability tracking  
- ❌ No proper conflict handling system  
- ❌ Poor accountability in resource usage  

---

## ✅ Our Solution

EduSpark solves these problems by implementing:

- ✔️ Centralized **Facilities & Assets Catalogue**  
- ✔️ Real-time availability tracking  
- ✔️ Booking workflow system (Pending → Approved/Rejected)  
- ✔️ Conflict prevention for overlapping bookings  
- ✔️ Role-based system (Admin / User)  
- ✔️ Clean and modern UI for better usability  

---

## ⚡ Why EduSpark is Different

- 🚀 Modern Smart Campus concept implementation  
- 🎯 Fully structured workflow system  
- 🔐 Secure and role-based access control  
- 💡 Designed based on real-world university problems  
- 📊 Scalable for future features (analytics, notifications)  

---

## 🏫 Module A – Facilities & Assets Catalogue

- Manage resources:
  - Lecture Halls
  - Labs
  - Meeting Rooms
  - Equipment (Projectors, etc.)

- Each resource includes:
  - Name
  - Type
  - Capacity
  - Location
  - Availability
  - Status (ACTIVE / OUT_OF_SERVICE)

- 🔍 Supports:
  - Search
  - Filtering
